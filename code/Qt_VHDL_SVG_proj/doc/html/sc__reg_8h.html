Detailed Description

SystemC module that implements a register and a counter.

This file contains the definition of the SystemC modules SyscReg and SyscCnt. The SyscReg module implements a register with a single data input and output. The register is synchronous, i.e., the output is updated on the rising edge of the clock signal. The SyscCnt module implements a counter with a single data output.

Both modules are implemented as template classes that can be instantiated with the following parameters:

  • W: width of the data bus (default is 1)
  • CLKEDGE: boolean value that specifies the clock edge to use (true for rising edge, false for falling edge). Default value is true.

The SyscReg and SyscCnt modules are derived from the SyscFlexInt module, which provides a flexible interface for the reset and enable signals of a module. The reset and enable signals can be active high or active low, and can be synchronous or asynchronous. The module can be configured to use the reset and enable signals or not, and can be configured to use the rising edge or falling edge of the clock signal.
