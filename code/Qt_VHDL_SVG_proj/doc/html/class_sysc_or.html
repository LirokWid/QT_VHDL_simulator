Detailed Description

template<unsigned int N = 2, unsigned int W = 1>
class SyscOr< N, W >

The SyscOr class.

A generic SystemC module that implements an OR gate with N inputs and 1 output. The gate is combinational, i.e., the output is updated whenever any of the inputs changes.
