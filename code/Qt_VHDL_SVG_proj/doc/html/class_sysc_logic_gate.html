Detailed Description

template<unsigned int N = 2, unsigned int W = 1>
class SyscLogicGate< N, W >

The SyscLogicGate::sc_core::sc_module class.

A generic SystemC module that implements a logic gate with N inputs and 1 output. The gate is combinational, i.e., the output is updated whenever any of the inputs changes.
