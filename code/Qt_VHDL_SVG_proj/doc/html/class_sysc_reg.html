Detailed Description

template<unsigned int W = 1, bool CLKEDGE = DEFAULT_CLKEDGE, flexinterface_t FLEXINT = DEFAULT_FLEXINT>
class SyscReg< W, CLKEDGE, FLEXINT >

The SyscReg class.

A generic SystemC module that implements a register with a single data input and output. The register is synchronous, i.e., the output is updated on the rising edge of the clock signal.
