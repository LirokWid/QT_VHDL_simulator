The \doxylink{class_sysc_xor}{Sysc\+Xor} class. 

A generic SystemC module that implements an XOR gate with N inputs and 1 output. The gate is combinational, \doxylink{class_sysc_add__pv_a57417948111cea94689e3dbc33fd51d1}{i.\+e}., the output is updated whenever any of the inputs changes.
