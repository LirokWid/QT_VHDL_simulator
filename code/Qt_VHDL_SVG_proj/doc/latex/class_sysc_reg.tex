The \doxylink{class_sysc_reg}{Sysc\+Reg} class. 

A generic SystemC module that implements a register with a single data input and output. The register is synchronous, \doxylink{class_sysc_add__pv_a57417948111cea94689e3dbc33fd51d1}{i.\+e}., the output is updated on the rising edge of the clock signal.
