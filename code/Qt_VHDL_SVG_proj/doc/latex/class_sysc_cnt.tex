The \doxylink{class_sysc_cnt}{Sysc\+Cnt} class. 

A generic SystemC module that implements a counter with a single data output. The counter is synchronous, \doxylink{class_sysc_add__pv_a57417948111cea94689e3dbc33fd51d1}{i.\+e}., the output is updated on the rising edge of the clock signal.
